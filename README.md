# Odin Recipes

## Description
This is the first project from the Foundations course of **The Odin Project**. The goal of this project is to demonstrate basic HTML skills, including creating web pages, using headings, paragraphs, lists, and images, as well as managing file structures and navigation between different HTML documents.

## Technologies Used
* HTML5

## Project Structure
The project is organized with a clear hierarchy for better management:
* `index.html`: The main landing page containing links to all recipes.
* `recipes/`: A directory containing individual recipe pages.
* Each recipe folder includes its own HTML file and related assets.

## Skills Learned
- Structuring a web project with multiple directories.
- Working with absolute and relative file paths.
- Displaying images and creating hyperlinks.
- Using Git and GitHub for version control.

## How to View
To view this project, clone the repository and open `index.html` in any web browser or Live Preview here: https://hoanggiabao123.github.io/odin-recipes/
